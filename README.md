# Club Maul Bot
A simple discord bot for Club Maul.

### Maul[root]/config.js
This houses your discord bot token and ID.

###Maul[root]/data/vrcLogin.json
This houses your vrchat service account login.
You acquire the 'qr code' token by enabling 2FA, scanning the 2FA QR code on the VRChat website, and then manually retrieving it from the link that pops up on your phone.
After that, you must complete the 2FA verification process normally.
